## STANDALONE BALANCE SHEET

This is a compositional donut-chart diagram with two side-by-side balance sheet breakdowns: Assets and Liabilities. It shows the proportional composition of each category for FY25 and FY24, with corresponding amount tables below. The relationships are compositional, showing each balance sheet category as parts of the whole.

- **Center node:** Assets
